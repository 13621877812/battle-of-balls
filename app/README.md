### 项目结构

```
.   
├── build               构建配置
│   ├── webpack.config.js
│   └── webpack.dev.js
├── dist                打包发布
├── package.json        依赖
├── src                 源文件目录
│   ├── api             数据接口
│   ├── assets          
│   │   ├── css			样式
│   │   └── utils		工具js
│   ├── index.html
│   └── main.js			入口js文件
└── README.md
```

### 开发

` npm install`

本地开发

`npm run dev`

打包

`npm run build`
